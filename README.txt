YouTube + Twitch 同時視聴ビューワー
============================

【推奨】 start.bat をダブルクリックで起動
  → ローカルサーバが立ち上がり、自動的にブラウザで開きます。
    YouTube / Twitch の埋め込みAPIが動作するため、一括再生・同期などの機能が使えます。
    Python 3 が必要です（未インストールならエラーが出ます）。

【簡易起動】 index.html をダブルクリック
  → file:// プロトコルで開かれるため、簡易モードになります。
    ・各動画は再生できます
    ・「全再生」「全ミュート」「シーク同期」などの一括操作は使えません
    ・各動画のプレーヤー上のコントロールで個別操作してください

使い方
------
1. ツールバーの「レイアウト」を選ぶと、その数だけ空の枠が表示されます。
2. 各枠の入力欄に URL / 動画ID を入れて「表示」を押すと、その枠に配信が表示されます。
   ・YouTube: youtu.be/... / youtube.com/watch?v=... / 動画ID
   ・Twitch : twitch.tv/チャンネル名 / twitch:チャンネル名
3. 空き枠の「💬 この枠にチャットを表示」を押すと、その枠がチャット枠になります。
   上部のメニューで、他の枠に表示している配信を選ぶと、そのチャットが表示されます。
   ・対応: YouTube ライブ / Twitch チャンネル
   ・チャット表示にはローカルサーバ（start.bat）での起動を推奨します。

困ったとき
----------
- 「動画が表示されない」 → Python が無い場合は https://www.python.org/ から
  Python 3 をインストールしてから start.bat を実行してください。
- 「埋め込み禁止」エラーが出る動画 → 著作権者や配信者が埋め込みを禁止している
  動画です。「元のページで開く」ボタンから直接視聴してください。

ショートカット
--------------
Space  : すべて再生 / 一時停止 切替
M      : 全ミュート / 解除 切替
R      : すべて先頭から再生
F      : 全画面切替
S      : ソロモード切替
1〜9   : レイアウト変更
Del    : フォーカス中の枠を空にする
